#### [[Вопросы к Экзамену]]
---
#### Основная теорема арифметики
$$\begin{align}
\forall a \in\mathbb{N } \ \exists \text{ разложение } a=\prod_{i=1}^n{p_i^{k_i}} \\
\text{где } p_i \text{— различные простые, } k_i \in \mathbb{N} \\
(\text{единственно с точностью до перестановки } p_i)
\end{align}$$
*Зам.* $a=1\ n=0\ a=\text{просто 1 множитель}$ 
$$\begin{align}
&\Box \ (\exists) \ \text{База: 1, простое число} \\
&\text{Переход: } a\to q \\
&a\not=1, \text{не простое }\quad \exists\ q: a\ \vdots \ q\ (q\not=a, q\not=1) \\
&r=\frac{a}{q} (r\not=a, r\not=1) \ a=qr, q<a, r < a \\ 
&| \Rightarrow \exists \text{ разложение}
\end{align}$$
$$\begin{align}
	\Box \ (!) \quad \quad &\\ 
	\text{ Лемма: } &xy \ \vdots \ z \\
	&(x, z) = 1 \Rightarrow y \ \vdots \ z \\
	xy \ \vdots \ p \in &\ \mathbb{P} \\
	x \ \vdots\not \ p \Rightarrow &y \ \vdots \ p \\
\end{align}$$
$$\begin{align}
	&\text{Пусть } \exists \ a: \text{2 разложения} \\
	&\text{Рассмотрим } min \ a \\
	&| \Rightarrow \text{в разложениях нет общих множителей} \\
	&a = p\cdot\ldots=q\cdot\ldots (\text{без } p, q\not=p)=q\cdot a' \\
	&\text{По лемме } q \ \vdots\not \ p \Rightarrow a'\ \vdots \ p \\
	&a'=p\cdot\ldots=\ldots(\text{без } p) \\
	&| \Rightarrow \text{2 разных разложения } a'<a
\end{align}$$
*Сл.*
$$a \in \mathbb{Z}: a = \pm\prod_{i=1}^n{p_i^{k_i}}$$
Причём разложение единственно с точностью до перестановки.

*Сл.*
$$\begin{align}
&a = p_1^{k_1}\cdot\ldots\cdot p_n^{k_n} \\
&p_i \text{ — простые}, k_i \in \mathbb{N}\cup\{0\} \\
&(\text{единственно с точностью до перестановки } p_i \text{и добавления/убирания } p_i^{k_i})
\end{align}$$

*Утв.*
$$\begin{align}
&a = p_1^{k_1}\cdot\ldots\cdot p_n^{k_n} \\
&d = p_1^{m_1}\cdot\ldots\cdot p_n^{m_n} \\
&a \ \vdots \ d \Leftrightarrow \forall i: k_i > m_i \\
&\Box \frac{a}{d} = \prod_{i=1}^n{p_i^{k_i-m_i}}, \text{если }  k_i\geq m_i \Rightarrow \frac{a}{d}\in\mathbb{N}\Rightarrow a\ \vdots\ d \\
&\text{если } \exists\ i: \ k_i<m_i \quad \frac{a}{d} = \frac{\ldots \ \vdots\not \ p_i}{p_i^{m_i-k_i}} \not\in\mathbb{N}
\end{align}$$

*Утв.*
$$\begin{align}
&a = p_1^{k_1}\cdot\ldots\cdot p_n^{k_n} \\
&b = p_1^{l_1}\cdot\ldots\cdot p_n^{l_n} \\
&| \Rightarrow (a,b)=\prod_{i=1}^n{p_i^{\min(k_i,l_i)}}
\end{align}$$

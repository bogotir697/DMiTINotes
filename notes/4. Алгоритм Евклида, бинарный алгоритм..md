#### [[Вопросы к Экзамену]]
---
### Описание Алгоритма Евклида
$$\begin{align*}
	&a,b\in\mathbb{Z}. \quad a > b \quad (\text{Зам. } a = r_{-2}, b=r_{-1})\\
	&a = b*q_0 + r_0 \\
	&b = r_0*q_1 + r_1 \\
	&r_0 = r_1*q_2 + r_2 \\
	&\quad \vdots \\
	&r_k = r_{k+1}*q_{k+2} + r_{k+2} \\
	&\quad \vdots \\
	&r_{n-2}=r_{n-1}*q_n+0 \quad\big(\text{Утв. }\exists n: r_n = 0\big)
\end{align*}$$
$|b|>r_0>r_1>r_2>\ldots$ — не может делиться $\infty$ 
*Утв.* Мн-во общих делителей $r_{k-1}$ и $r_{k-2}$ равно мн-ву общих делителей $r_{k-1}$ и $r_{k}$
*Теорема.* $r_n=0$
$r_{n-1}$ — общий делитель $a$ и $b$, делящийся на их остальные общие делители
$\Box \text{ у пар } (a,b) \text{ и } (r_{n-2},r_{n-1}) \text{ одни и те же общие делители } r_{n-2}\ \vdots\ r_{n-1}$ .
### Бинарный алгоритм Евклида
```c
a,b
d = 1
while (a % 2 == 0) and (b % 2 == 0) {
	a = a / 2
	b = b / 2
	d = d * 2
}
while (a != b) {
	while (a % 2 == 0) a = a / 2
	while (b % 2 == 0) b = b / 2
	if (a > b) a = a - b
	else if (b > a) b = b - a
}
d = d * a
```

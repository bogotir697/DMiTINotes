#### [[Вопросы к Экзамену]]
---
*Опр.* Делимость
$$ \begin{align}
	a\in \mathbb{Z} \ \vdots \ b\in\mathbb{Z}: &\\
	\exists \ c \in \mathbb{Z}: \ \ a &= b \cdot c
\end{align}
$$
*Свойства:*
1) $\forall a\in\mathbb{Z} \quad a \ \vdots \ a$ 
2) $a \ \vdots \ b, b \ \vdots \ a \Rightarrow a = \pm b$ 
	- $a=bc=akc\Rightarrow kc=1 \Rightarrow k = c = \pm 1$
3) $a_1 \ \vdots \ b, a_2 \ \vdots \ b \Rightarrow a_1 \pm a_2 \ \vdots \ b$ 
	- $a_1 = bc_1, a_2 = bc_2 \Leftrightarrow a_1 \pm a_2 = bc_1\pm bc_2 = b(c_1\pm c_2)$ 
4) $a_1 \ \vdots \ b \text{ or } a_2 \ \vdots \ b \Rightarrow a_1 a_2 \ \vdots \ b$ 
5) *Транзитивность*. $a \ \vdots \ b, b \ \vdots \ d \Rightarrow a \ \vdots \ d$ 
	- $a = bc = dkc = d(kc)$ 
6) $a=bq+r, a \ \vdots \ d, b \ \vdots \ d \Leftrightarrow b \ \vdots \ d, r \ \vdots \ d$ 
*Опр.* НОД
$$ \begin{align}
a, b, d \in \mathbb{Z} \\
(a, b) = d \Leftrightarrow \begin{cases} a = d\cdot c_1 \\ b = d\cdot c_2 \end{cases}, \quad c_1 \ \vdots\not \ c_2

\end{align}
$$
*Опр.* Числа $a \in \mathbb{N}$ и $b \in \mathbb{N}$ называются *взаимно простыми*, если $(a,b)=1$.
*Опр.* Число  $p \in \mathbb{N}$ называется *простым*, если у него ровно 2 $\mathbb{N}$ делителя. $(p \text{ и } 1)$ 
	*Зам.* 1 — не простое
